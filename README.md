# PocketMLO

Serves `pocketmlo (4).html` at the site root.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build for production

```bash
npm run build
npm start
```

## Deploy

Deploy the repo to any Node host (Vercel, a VPS, etc.). The HTML file is read from the project root at runtime.
